# A Says Hi to B

## Problem

Take two names **A** and **B** as input and print:

```
A says Hi to B
```

## Problem Constraints

```
1 ≤ len(A), len(B) ≤ 15
```

Both names consist of lowercase English letters.

## Input Format

Two input lines:

Line 1 → Name **A**
Line 2 → Name **B**

## Output Format

Print the following sentence:

```
A says Hi to B
```

## Example Input

```
Ram
Shyam
```

## Example Output

```
Ram says Hi to Shyam
```

## Approach

Read the two input strings and concatenate them with the required message.

## Time Complexity

O(1)

## Space Complexity

O(1)
