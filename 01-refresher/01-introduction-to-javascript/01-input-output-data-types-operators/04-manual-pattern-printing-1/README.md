# Manual Pattern Printing - 1

## Problem

Print the following pattern:

```
*****
****
***
**
*
```

## Output Format

Print 5 lines with decreasing stars.

```
*****
****
***
**
*
```

## Example Output

```
*****
****
***
**
*
```

## Approach

The number of stars decreases by one in every line.

Line 1 → 5 stars
Line 2 → 4 stars
Line 3 → 3 stars
Line 4 → 2 stars
Line 5 → 1 star

This can be implemented using loops.

## Time Complexity

O(n²)

## Space Complexity

O(1)
