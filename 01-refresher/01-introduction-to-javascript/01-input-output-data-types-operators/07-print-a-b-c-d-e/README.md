# Print A B C D E

## Problem

Print the first five letters of the English alphabet.

```
A
B
C
D
E
```

## Output Format

Each letter must be printed on a new line.

## Example Output

```
A
B
C
D
E
```

## Approach

Simply print the characters **A, B, C, D, E** on separate lines.

## Time Complexity

O(1)

## Space Complexity

O(1)
