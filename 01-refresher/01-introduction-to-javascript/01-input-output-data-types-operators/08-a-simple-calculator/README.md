# A Simple Calculator

## Problem

Given two integers **A** and **B**, print the following operations in order:

```
A + B
A * B
A - B
A / B
```

## Problem Constraints

```
1 ≤ A, B ≤ 10⁴
```

## Input Format

Two input lines:

Line 1 → Integer **A**
Line 2 → Integer **B**

## Output Format

Print the results separated by spaces in a single line.

```
A+B A*B A-B A/B
```

## Example Input

```
4
5
```

## Example Output

```
9 20 -1 0
```

## Explanation

```
4 + 5 = 9
4 * 5 = 20
4 - 5 = -1
4 / 5 = 0
```

Division uses integer division.

## Approach

Read the input numbers and perform the four arithmetic operations in the specified order.

O(1)

## Space Complexity
